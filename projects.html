Past Projects

Masters Project

Master of Engineering project was implementing and aiding in additional research under a PhD candidate in the Communication and Collaborative Technologies Lab under Susan Fussell . Project was a collaborative effort with other M.Eng candidates, designing NLP features to implement published research on discovering signals of formality/politeness in the English language. In addition, my portion included using neural/probabilistic machine learning models to analyze new datasets on university students of both native English and non-native English speaking backgrounds, using their perceptions of politeness in textual English data.

Runtime Verification

Under the guidance of a PhD course on runtime verification, designed a final project for analyzing C memory allocation in Linux kernels. The project involved designing specifications inspired by published research on C memory safety and attempted to provide a language-agnostic approach for runtime verification. The implementation focused on SystemTap and was tested on the DaCapo suite of Java projects.

Web App Development

As part of a group project in Spring 2020, I was a main contributor to both front-end and back-end development of a web app Subreddit Recommender. Built in Flask and deployed in Heroku, it uses a specialized dataset of Reddit posts to recommend subreddit communities given a user's post. Techniques employed include cosine similarity measures, sentiment analysis and pseudo-relevance feedback.


Computational Linguistics

My first project in computational linguistics was in Spring 2018. Using the Foma FST for parsing grammar via a state machine and Java to store strings from the French Gigaword Third Edition linguistic corpus, I implemented a French-English Machine Translation engine for French statements and questions taken from the corpus.

During Fall 2019, I took leadership roles in group projects for Natural Language Processing. Implementations of projects included PyTorch representations of neural networks and other ML methods, exploring Naive Bayes, n-gram models, feed-forward neural networks, and recurrent neural networks for tasks such as spam filtering, parts-of-speech tagging, and sentiment analysis. Final project included integrating and replicating some of the latest research in pretrained models.The project used both state-of-the-art BERT models and more traditional approximations for common-sense reasoning tasks, using data from the Cloze test corpus ROCStories.

In Spring 2020, for the purpose of language typology and exploring new methods in Predictive Areality Theory, I implemented feed-forward neural networks in PyTorch to predict new linguistic areas. These networks were trained with data from the World Atlas of Language Structures. These PAT models were an experiment for using existing, publicly-available data for this new area of linguistic research.


Operating Systems

For Systems Programming in Fall 2017, the final project was my own implementation of the <malloc> library. This included implementation of page allocation and memory management, via a linked-list of blocks stored within a 4KB page.

During Fall 2018, as part of the Practicum in Operating Systems, when given an incomplete Unix-based operating system, I implemented in C several missing data structures for executing basic functionality. This included a simple linked-list queue, a user-level thread control unit, a multi-level feedback queue for process scheduling, a write-through cache with a CLOCK eviction algorithm, and a FAT file system.
